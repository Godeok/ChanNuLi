## 1. 프로젝트 설명
찬누리는 노인 건강관리 앱으로 노인분들이 스스로 건강 상태를 진단하고, 다양한 질병과 건강 정보를 얻고, 간단한 게임을 하면서 치매를 예방할 수 있습니다. 노인분들이 어플을 쉽게 사용하실 수 있도록 간편한 화면 구성과 조작 방법 또한 제공합니다.
